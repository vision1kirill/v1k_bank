# Инструкция по настройке и запуску бота

## Шаг 1: Получить Telegram Bot Token (если ещё нет)

1. Открой Telegram → найди `@BotFather`
2. Напиши `/newbot`
3. Придумай имя и username боту
4. Скопируй токен вида `7890123456:AAFxxxxxxxxxxxxxx`

---

## Шаг 2: Получить T-Bank Invest API токен

1. Открой приложение **Т-Инвестиции** на телефоне
2. Профиль (правый нижний угол) → **Настройки** → **Прочее**
3. Нажми **"Подключить API"**
4. Создай новый токен — выбери **"Полный доступ"** (нужен для торговли)
5. Скопируй токен — он показывается **один раз**!

> ⚠️ Храни токен в секрете. Не публикуй его в GitHub.

---

## Шаг 3: Узнать свой Telegram User ID

1. Напиши боту `@userinfobot` в Telegram
2. Скопируй цифры из поля `Id`

---

## Шаг 4: Деплой на Railway

### 4.1 Создай репозиторий GitHub

```bash
cd investment-bot
git init
git add .
git commit -m "Initial investment bot"
```

Создай репозиторий на github.com и запушь:
```bash
git remote add origin https://github.com/ВАШ_НИК/investment-bot.git
git push -u origin main
```

### 4.2 Создай проект на Railway

1. Зайди на [railway.app](https://railway.app)
2. **New Project** → **Deploy from GitHub repo**
3. Выбери свой репозиторий `investment-bot`
4. Railway автоматически найдёт `Dockerfile`

### 4.3 Добавь переменные окружения

В Railway → Settings → Variables добавь:

| Переменная | Значение |
|---|---|
| `TELEGRAM_TOKEN` | Твой токен от BotFather |
| `TINKOFF_TOKEN` | Твой токен от Т-Инвестиций |
| `ALLOWED_USER_IDS` | Твой Telegram User ID (например: `123456789`) |
| `USE_SANDBOX` | `true` (сначала тест!) или `false` (реальная торговля) |
| `TINKOFF_ACCOUNT_ID` | Оставь пустым — бот возьмёт первый счёт |

### 4.4 Запусти деплой

Railway автоматически задеплоит бот. Через 1-2 минуты он будет запущен.

---

## Шаг 5: Первый запуск

1. Найди своего бота в Telegram по username
2. Нажми `/start`
3. Сначала проверь в **sandbox режиме** (`USE_SANDBOX=true`)
4. Убедись что всё работает корректно
5. **Только тогда** переключи на `USE_SANDBOX=false` для реальной торговли

---

## Использование бота

### Создание DCA стратегии
1. Нажми **📊 Стратегии** → **➕ Новая стратегия**
2. Выбери **📈 DCA (Усреднение)**
3. Введи тикер (например: `SBER`)
4. Введи сумму одной покупки (например: `3000`)
5. Выбери частоту: еженедельно или ежемесячно
6. Выдели бюджет (например: `30000`)
7. Подтверди создание

### Создание Grid стратегии
1. **📊 Стратегии** → **➕** → **🕸️ Grid**
2. Тикер → нижняя граница → верхняя граница → шаг → сумма на уровень → бюджет
3. Бот автоматически выставит лимитные ордера

### Создание стратегии дивидендов
1. **📊 Стратегии** → **➕** → **♻️ Реинвест дивидендов**
2. Введи тикеры через пробел: `SBER LKOH GAZP`
3. Укажи бюджет

### Ежедневный анализ
Каждый день в **10:00 МСК** бот автоматически отправит:
- Анализ топ-12 инструментов с сигналами (ПОКУПАТЬ / ДЕРЖАТЬ / ПРОДАВАТЬ)
- Персональную сводку по твоим позициям (P&L, рекомендации)

### Еженедельный отчёт
Каждый **понедельник в 11:00 МСК** — сводный отчёт по всем стратегиям.

---

## Важные замечания

- **Бот торгует только выделенным бюджетом** — деньги вне стратегий не трогает
- **Sandbox режим** — безопасно тестировать, деньги не реальные
- **Реальная торговля** — убедись что понимаешь риски
- Бот работает 24/7 на Railway
- При ошибках/сбоях Railway автоматически перезапустит бота

---

## Поддерживаемые тикеры (для анализа рынка)

SBER, LKOH, GAZP, NVTK, ROSN, YDEX, MGNT, MTSS, TATN, GMKN, TMOS, EQMX

Для стратегий поддерживается **любой тикер MOEX** (введи тикер — бот найдёт автоматически).
